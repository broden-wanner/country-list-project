# Tech Stack

This page outlines all the technologies used in the project and their purposes.

## Backend

- [FastAPI](https://fastapi.tiangolo.com/): Backend framework and API doc generation.
- [NetworkX](https://networkx.org/): Graph operations.

## Deployment

- [Docker](https://www.docker.com/): Containerization.
- [Nginx](https://www.nginx.com/): Server.
- [Digital Ocean](https://www.digitalocean.com/): Server provider.
- [GitHub Actions](https://github.com/features/actions): Continuous Integration.

## Testing

- [pytest](https://docs.pytest.org/en/6.2.x/): Unit testing.
- [requests](https://docs.python-requests.org/en/latest/): Functional testing.
- [Codecov](https://about.codecov.io/): Code coverage reporting.

## Developer Activities

- [MkDocs](https://www.mkdocs.org/): Documentation generation with MarkDown.
- [flake8](https://flake8.pycqa.org/en/latest/): Linting.
- [black](https://github.com/psf/black): Formatter.
- [mypy](http://mypy-lang.org/): Static type checking.
- [Poetry](https://python-poetry.org/): Package management.
